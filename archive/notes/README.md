# Archive: Notes

This folder is used for preserving old ITSM development notes, documentation, and related materials for future reference during the transformation to the HRMS (Human Resource Management System).

## Purpose
- Stores deprecated development notes from the original ITSM system
- Maintains reference documentation and technical notes
- Documents the evolution of development practices and decisions

## Note
Files in this directory are archived and should not be modified or used in active development.
