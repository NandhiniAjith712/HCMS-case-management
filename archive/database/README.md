# Archive: Database

This folder is used for preserving old ITSM database schemas, migrations, and related files for future reference during the transformation to the HCMS Case Management system.

## Purpose
- Stores deprecated database schemas from the original ITSM system
- Maintains reference migration scripts and SQL files
- Documents the evolution of database structure and design

## Note
Files in this directory are archived and should not be modified or used in active development.
