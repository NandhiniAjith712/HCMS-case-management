# Archive: Frontend Modules

This folder is used for preserving old ITSM frontend module functionality for future reference during the transformation to the HCMS Case Management system.

## Purpose
- Stores deprecated frontend module implementations from the original ITSM system
- Maintains reference modular frontend components and integrations
- Documents the evolution of frontend module architecture

## Note
Files in this directory are archived and should not be modified or used in active development.
