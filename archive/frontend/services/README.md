# Archive: Frontend Services

This folder is used for preserving old ITSM frontend service layer for future reference during the transformation to the HCMS Case Management system.

## Purpose
- Stores deprecated frontend service implementations from the original ITSM system
- Maintains reference API client services and data fetching logic
- Documents the evolution of frontend service architecture

## Note
Files in this directory are archived and should not be modified or used in active development.
