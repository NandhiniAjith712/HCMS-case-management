# Notifications

HCMS sends notifications to employees about updates to their cases.

## When employees receive notifications

- A new case is created.
- The status of a case changes.
- A comment or message is added to the case.
- The case is escalated.
- An HR user requests more information.
- An escalation consent request is created.
- The case is resolved or closed.

## Where notifications appear

Notifications appear on the employee dashboard and in the notification panel. Employees can mark notifications as read individually or mark all as read at once.

## Notification types

- In-app notifications appear inside the HCMS portal.
- Email notifications may also be sent depending on system configuration.

## Backfill

If the system has old notifications that were not generated earlier, administrators can run backfill operations. Employees do not need to worry about this.
