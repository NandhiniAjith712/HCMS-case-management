# Dashboard

The employee dashboard shows a summary of the employee's own cases.

Metrics shown on the dashboard cards:
- Total: all cases raised by the employee.
- Open: cases with status `new` (awaiting action).
- In Progress: cases currently being worked on by HR.
- Closed: cases that have been completed or confirmed and closed.

The dashboard also shows:
- Recent cases (last 5).
- Notifications related to the employee's cases.

All dashboard data is filtered to the logged-in employee only. Employees cannot see other employees' cases.
