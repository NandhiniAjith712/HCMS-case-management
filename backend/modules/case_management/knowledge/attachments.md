# Attachments

Employees can attach files to a case when creating it or later when adding a comment or message.

## Supported file types

Images, PDFs, documents, spreadsheets, text files, and zip archives are generally supported. The exact allowed types are configured by the system.

## File size limit

The default upload limit is 10 MB per file. Larger files may be rejected.

## How to add attachments

- When creating a case, use the attachment field to select or drag and drop files.
- When viewing a case, use the comment or message section to attach files.
- Multiple files can be uploaded at once.

## Who can see attachments

Attachments are visible to the employee and the assigned HR users who have permission to view the case. For special cases (confidential, sensitive, anonymous), visibility follows the same rules as the case itself.

## Downloading attachments

Employees can download attachments from their own cases by clicking the download link on the case detail page.
